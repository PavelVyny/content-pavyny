    1. Create `web/src/lib/types.ts` with shared TypeScript types:
       ```typescript
       export interface HookVariant {
         variant: string; // "A", "B", "C"
         visual: string;
         voiceover: string;
       }

       export interface ScriptBeat {
         id: number;
         scriptId: number;
         order: number;
         visual: string;
         voiceover: string;
         duration: string | null;
       }

       export interface AntiSlopScore {
         directness: number;
         rhythm: number;
         trust: number;
         authenticity: number;
         density: number;
         total: number;
         notes: string;
       }

       export interface Script {
         id: number;
         title: string;
         format: string;
         status: "generating" | "draft" | "ready" | "recorded";
         hooks: HookVariant[] | null;
         selectedHook: string | null;
         titles: string[] | null;
         thumbnail: string | null;
         durationEstimate: string | null;
         antiSlopScore: AntiSlopScore | null;
         devContext: string | null;
         createdAt: Date;
         updatedAt: Date;
         beats?: ScriptBeat[];
       }

       export interface VideoFormat {
         name: string;
         slug: string;
         description: string;
         beatCount: number;
       }
       ```

    2. Create `web/src/lib/db/schema.ts` with Drizzle schema — exactly as specified in research:
       - `scripts` table with JSON columns for hooks, titles, antiSlopScore using `text("col", { mode: "json" }).$type<T>()`
       - `beats` table with foreign key to scripts (onDelete: cascade), order column, visual/voiceover text columns
       - Status field with enum: "generating", "draft", "ready", "recorded"
       - Use `integer("col", { mode: "timestamp" })` for createdAt/updatedAt with `$defaultFn(() => new Date())`

    3. Create `web/src/lib/db/index.ts` — singleton DB connection:
       - Path: `path.join(process.cwd(), "data", "scripts.db")`
       - Enable WAL mode: `sqlite.pragma("journal_mode = WAL")`
       - Enable foreign keys: `sqlite.pragma("foreign_keys = ON")`
       - Module-level singleton pattern (survives hot reload)
       - Export `getDb()` function

    4. Create `web/drizzle.config.ts`:
       ```typescript
       import { defineConfig } from "drizzle-kit";
       export default defineConfig({
         schema: "./src/lib/db/schema.ts",
         out: "./drizzle",
         dialect: "sqlite",
         dbCredentials: { url: "./data/scripts.db" },
       });
       ```

    5. Push schema to database:
       ```
       cd web && npx drizzle-kit push
       ```
       This creates the scripts.db file and both tables.
