# Video Metrics Log

## Instructions

After publishing each video, wait 48 hours, then log metrics below.

**Where to find metrics:** YouTube Studio > Content > Click video > Analytics > Overview tab.

**Fields:**
- **#:** Sequential video number
- **Title:** Video title as published
- **Date:** Publish date (YYYY-MM-DD)
- **Format:** Which format template was used (The Bug, The Satisfaction, Before/After, The Decision, The Trick, The Fail, The Number)
- **Hook Type:** What the first 3 seconds showed/said
- **Duration:** Video length in seconds
- **Views:** Total views at 48h mark
- **Engaged Views:** Views where viewer watched past the hook (YouTube Studio > Engagement tab)
- **Avg Duration:** Average view duration in seconds
- **Stayed %:** Percentage of viewers who watched to end (YouTube Studio > Retention tab)
- **Swiped %:** Percentage who swiped away (YouTube Studio > Retention tab)
- **Subs Gained:** New subscribers from this video (YouTube Studio > Audience tab)

## Pattern Notes

**From historical data (6 videos):**
- Problem-solution formats (The Bug, The Trick) outperform showcase by 2-3x on views and retention
- Short, personality-driven titles beat generic "Episode N" by 3-4x on views
- Videos with planned visual sequences (dual-track scripts) correlate with higher retention
- Highest sub conversion: #3 (+21 from 7.7K views = 0.27%) used The Bug format
- Highest retention: #6 (75.7%) used The Trick format with a game reference hook (TLOU)
